# Card Component
 
* Utilise les "Variant" normalisées
* Présente un effet de zoom au survol ("mouseover")
* Accepte un "title" pour son bandeau de titre.
* Accepte une une structure HTML pour le corps de la card
* "collapsible" s'ouvre et se ferme par un clique sur le bandeau de titre


## Usage

```html
<ui-card title="STRING" variant="VARIANT">
    <!--HTML CONTENT-->
</ui-card>

```
